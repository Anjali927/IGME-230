UNIX Vocabulary

pwd
Prints the full system path of the current working directory.

man commandName
Can format and display the user manual pages.

cp file1 file2
Creates an exact copy of file1 under the new name: file2.

mv file1 file2
Gives file1 the new name, file2.

ls
Used to list the computer files and directories under the current working directory.

ls -al or ls -la
Provides a long list of all files, including hidden files, and details within each file.

ls -l
Generates a long listing of a directory, file, and file permissions.

cd folder1
Changes the current directory to folder1.

cd ..
Used to move up a directory, or change to the parent of the current directory.

cd /
Changes the current directory to the root directory.

cd
Used to change directory. When used on its own, the cd command returns the user to the previous current directory.

chmod 755 folder1
Set folder permissions to 755. Means read and execute access for everyone and write access for the owner of the folder.

chmod 644 file1
Set file permissions to 644. Means read and write access for the owner of the file and read access for everyone else.

chmod 700 folder1
Set folder permissions to 700. Means owner has all access to the folder and everyone else has no access.

mkdir folder1
Creates a directory called folder1.

rmdir folder1
Removes the directory folder1.

rm file1
Will remove file1.

rm proj?.html
Deletes all files in the form proj?.html where the ? indicates any single character.

rm*
Selects all files and deletes them.

cal
Displays a calendar in the terminal.

cal 5 2016
Displays a full calendar of May 2016 in the terminal.

cat file1
Will display the contents of file1.

cat file1 file2 >> file3
Will copy file1 and file2 into file3.

echo "goop" >> file3
Displays the string "goop" that is passed in as the argument. Redirects "goop" to file3.

nano file1.txt
Opens a text file in the text editor Nano, titled file1.

w or who
Shows who is logged on, and what they are doing.

finger userName
Can be used to find out information about a computer user. May reveal login name, full name, and other details about the specific user.

write userName
Sends a message to another user. Copies lines from one's terminal to another user's terminal.

wc -w < file
Prints the number of words in the file.

wc -l < file
Prints the number of lines in the file.

wc -c < file
Prints the number of bytes in the file.
